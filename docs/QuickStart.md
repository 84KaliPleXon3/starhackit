# Quick Start

## Clone the source code

To get the latest code, grab it by cloning the repository from GitHub:

    $ git clone https://github.com/FredericHeem/starhackit.git yourproject
    $ cd yourproject
    
## Backend

    # cd server
    # npm install
    # npm run setup
    # npm start

## Frontend

    # cd client
    # npm install
    # npm start
